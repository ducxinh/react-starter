# React + TypeScript + Vite

## Installation
```bash
yarn install
```

## Run development
```bash
yarn dev
```

## Result
Open browser and navigate to [http://localhost:5173](http://localhost:5173)

## Run Lint
```bash
yarn lint
```

## Run Lint format
```bash
yarn lint:prettier
yarn fix:prettier
```

## Run Test
```bash
yarn test
```

## Build
```bash
yarn build
```
